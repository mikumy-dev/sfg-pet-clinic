  • About index.html

    th:replace="~{fragments/layout :: layout (~{::body},'home')}"

    th:replace To include a fragment template into index page, use th:fragment="layout" to define a fragment named "layout"
    ~{ } Don't know. Delete this and the th:replace="fragments/layout :: layout (~{::body},'home')" still works well.
    fragments/layout look at the path fragments/layout.html
    :: templatename::domselector, like <div th:replace="footer :: copy"></div>, replace the div with the "copy" fragment and the tag name is footer
    layout (~{::body},'home') (~{::body},'home'), the arguments of the fragment layout, parameter 1: the body element of the index.html, parameter 2: the 'home' string. ::body, look at the body element in current page
